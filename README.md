# 빙동댕 겨울 프로모션 랜딩페이지

빙동댕 빙수 프랜차이즈의 겨울 특별 프로모션을 위한 단일 페이지 랜딩페이지입니다.

## 🎯 프로젝트 목적

- **주제**: 빙동댕 빙수 프랜차이즈 겨울 프로모션
- **목적**: 매장 방문 활성화 및 신메뉴 홍보
- **타겟**: 동영상 광고를 통해 유입되는 모바일 중심 고객

## 🍧 프로모션 상품

### 1. 뜨거운 흑임자 빙수 (NEW)
- 따뜻한 흑임자죽과 차가운 빙수의 혁신적인 조합
- 겨울에도 즐길 수 있는 건강한 디저트
- SNS 인증 시 20% 할인 쿠폰 제공

### 2. 빙수 케이크
- 한국 최초 빙수케이크 포지셔닝
- 티라미수 스타일의 프리미엄 디저트
- **매장 예약 시 30% 할인** (35,000원 → 24,500원)
- 전용 보냉백 제공으로 신선도 유지

### 3. 콤보 프로모션
- 빙수케이크 예약 + 뜨거운 흑임자 빙수 미니 무료
- 또는 두 메뉴 동시 구매 시 35% 할인

## 🚀 기능

### 인터랙티브 요소
- ⏱️ **실시간 카운트다운**: 프로모션 종료까지 남은 시간 표시
- 🏪 **매장 필터링**: 전체/서울/경기/인천으로 지역별 필터
- 📞 **원클릭 전화**: 모바일에서 탭하여 바로 통화
- ❓ **FAQ 아코디언**: 클릭하여 답변 펼치기/접기
- 🎬 **스크롤 애니메이션**: 요소가 뷰포트에 들어올 때 페이드인
- 🔗 **스무스 스크롤**: 섹션 간 부드러운 이동

### 매장 데이터
현재 10개 매장 등록:
- 서울: 5개 (강남, 홍대, 명동, 신촌, 잠실)
- 경기: 3개 (분당, 수원, 일산)
- 인천: 2개 (구월, 송도)
